# Fill missing operation summaries

Fill missing operation summaries

## Usage

``` r
.paths_fill_summary(operation_summaries, endpoints, methods)
```

## Arguments

- operation_summaries:

  (`character`) Operation summaries from the API definition.

- endpoints:

  (`character`) Endpoint paths paired with operations.

- methods:

  (`character`) HTTP methods paired with endpoints.

## Value

(`character`) Human-readable operation summaries.
