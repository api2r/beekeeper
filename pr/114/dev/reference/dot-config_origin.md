# Extract the origin from an api definition

Extract the origin from an api definition

## Usage

``` r
.config_origin(api_definition)
```

## Arguments

- api_definition:

  ([`rapid::class_rapid`](https://rapid.api2r.org/reference/class_rapid.html))
  The API definition to generate package code from.

## Value

(`list`) The `url`, `format`, and `version` (if known) of the API
definition.
