# Fill missing operation identifiers

Fill missing operation identifiers

## Usage

``` r
.paths_fill_operation_id(operation_ids, endpoints, methods)
```

## Arguments

- operation_ids:

  (`character`) Operation identifiers from the API definition.

- endpoints:

  (`character`) Endpoint paths paired with operations.

- methods:

  (`character`) HTTP methods paired with endpoints.

## Value

(`character`) Operation identifiers.
