# Format one security argument help entry

Format one security argument help entry

## Usage

``` r
.security_arg_description_clean(security_arg_description, security_arg_name)
```

## Arguments

- security_arg_description:

  (`character(1)`) The argument description.

- security_arg_name:

  (`character(1)`) The argument name.

## Value

(`list`) A named list with `name` and `description`.
