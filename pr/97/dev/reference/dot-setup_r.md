# Set up package directories and dependencies

Set up package directories and dependencies

## Usage

``` r
.setup_r(pkg_dir, include_stbl = FALSE)
```

## Arguments

- pkg_dir:

  (`character(1)` or `fs_path`) The directory containing package files.

- include_stbl:

  (`logical(1)`) Whether to add `stbl` to Imports.

## Value

`NULL` (invisibly).
