# Append text conditionally

Append text conditionally

## Usage

``` r
.paste0_if(original, test, addition)
```

## Arguments

- original:

  (`character`) The original text.

- test:

  (`logical`) A condition for each element of `original`.

- addition:

  (`character`) Text to append when `test` is `TRUE`.

## Value

(`character`) The updated text.
