# Compile parameter class descriptions

Compile parameter class descriptions

## Usage

``` r
.compile_param_class_descriptions(type, allow_empty, required)
```

## Arguments

- type:

  (`data.frame`) Joined parameter type metadata.

- allow_empty:

  (`logical`) Whether empty parameter values may be represented as
  `NULL`.

- required:

  (`logical`) Whether each parameter is required.

## Value

(`character`) Parameter class descriptions.
