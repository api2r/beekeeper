# Fill a missing security scheme description

Fill a missing security scheme description

## Usage

``` r
.security_scheme_description_fill(
  security_scheme_description,
  security_scheme_type
)
```

## Arguments

- security_scheme_description:

  (`character(1)`) A security scheme description from the API
  definition.

- security_scheme_type:

  (`character(1)`) The security scheme type.

## Value

(`character(1)`) A security scheme description.
